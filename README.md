# Sprint 2 - Care Plus 2026 - Python

## Descrição do projeto

O sistema foi desenvolvido em Python com o objetivo de incentivar hábitos saudáveis através de um sistema simples de missões.

A aplicação funciona via terminal e permite que o usuário crie uma conta, faça login e interaja com missões do dia a dia, como beber água ou se exercitar. O usuário pode visualizar missões disponíveis, completá-las e acompanhar seu progresso.

O projeto foi desenvolvido utilizando Python puro, com foco em lógica de programação, uso de listas e dicionários, e criação de um menu interativo simples e funcional.

## Instruções de Uso

### Como testar a aplicação:

1. Execute o arquivo principal (`main.py`).
2. Escolha a opção **1** para criar uma conta.
3. Digite um usuário e uma senha.
4. Escolha a opção **2** para fazer login.
5. Após logar, utilize as opções do menu:
   - **3** para listar missões disponíveis e completas.
   - **4** para completar uma missão.
6. Digite **0** para sair do sistema.

---

### Estrutura de arquivos importante:

- `main.py` — responsável pelo menu principal e interação com o usuário.
- `funcoes.py` — contém as funções de cadastro, login e gerenciamento de missões.
